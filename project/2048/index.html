How to play: Use your arrow keys to move the tiles. When two tiles with the same number touch, they merge into one!

RUN THE AI: Click the "Run Trial" button to initiate the Expectimax Search Algorithm, and watch 2048 solve itself.

Note: This site is not the official version of 2048. It is an Artificial Intelligence project from Northeastern University. You can find the original version of the game at via http://git.io/2048. It is playable on desktop or mobile.
